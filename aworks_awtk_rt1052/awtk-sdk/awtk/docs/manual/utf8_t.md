## utf8\_t
### 概述
wchar_t和char类型转换接口。

示例：

```c
const char* str = "hello";
const wchar_t* wstr = L"hello";
char res_str[128];
wchar_t res_wstr[128];

tk_utf8_to_utf16(str, res_wstr, ARRAY_SIZE(res_wstr));
tk_utf8_from_utf16(wstr, res_str, ARRAY_SIZE(res_str));
```

#### tk\_utf8\_dup\_utf16 函数
