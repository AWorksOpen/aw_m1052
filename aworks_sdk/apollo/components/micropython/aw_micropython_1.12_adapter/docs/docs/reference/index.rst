MicroPython 语言及实现
=======================================

MicroPython 旨在实现 Python 3.4 标准的语言语法 （具有从更高版本中选择的功能）， 并且 MicroPython 的大多数功能与 `docs.python.org <https://docs.python.org/3/reference/index.html>`_ 中的 "Language Reference"文件中所述功能相同。


在 :ref:`corresponding chapter <micropython_lib>` 中介绍了 MicroPython 标准库。在 :ref:`corresponding chapter <cpython_diffs>` 章节描述了 MicroPython 和 CPython 之间的区别（主要涉及标准库和类型 ，也涉及语言级别的功能）。

本章介绍MicroPython实现的功能和特点以及他们的最佳使用方法。
