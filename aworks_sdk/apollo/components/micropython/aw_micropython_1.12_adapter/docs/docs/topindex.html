此文档仅适用于与MicroPython V1.12版本

MicroPython是近年国内外开源硬件（也是智能硬件）中最热门的主题之一，它使用Python语言在嵌入式中编程，不需要了解底层寄存器、数据手册、厂家的库函数，大部分外设和常用功能都有自己的库，使开发和移植变得容易和快速。

AWorks 是我司推出的跨平台的嵌入式操作系统，宗旨是“软件定义一切”，使应用和硬件平台彻底分离。为开发者提供的是高度抽象的通用接口，基于 AWorks 的软件与底层硬件无关，将开发者从“自底层寄存器开始开发、学习各种协议栈”的苦海中解救出来，使开发者可以回归产品本质，以应用为中心，将主要精力集中在需求、算法和用户体验等业务逻辑上。

AWorks-MicroPython是将Micropython移植到AWorks系统后的产物，此作为AWorks的一个可选组件，适配了AWorks上层标准API，具有统一的函数标准，只要可以运行AWorks的硬件平台，在资源允许的情况下都可运行AWorks-MicroPython。

本文将介绍AWorks-MicroPython的特点，及使用方法。
